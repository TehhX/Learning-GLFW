# Utilities
This folder contains meta utilities for sub-project(s).

## Embed
Embeds glsl (.frag, .vert) files into .c, .h files for use with sub-projects. Such sub-projects must have their CMake reconfigured with any changes to the glsl files.

## PointsAboutOrigin
Prints a given amount of 2D vertices about a given origin. Higher vertex counts approach a perfect circle.
